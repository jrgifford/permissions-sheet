Hello, world. This page will be the page about chmod

  • chmod 700 filename.txt. Only you can read or write to filename.txt.
  • chmod 744 filename.txt.. Only you can read, write or execute filename.txt, but everyone can read filename.txt.
  • chmod 444 filename.txt. Everyone can only read filename.txt.
  • chmod 777 filename.txt. This should never be used. It gives everyone read, write and execute access on filename.txt. For more on why this is a bad reason, look over here.
